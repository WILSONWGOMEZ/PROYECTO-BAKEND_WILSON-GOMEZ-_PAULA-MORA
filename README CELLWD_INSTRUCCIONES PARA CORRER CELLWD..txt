Readme CEELWD. 

Inicialmente debemos correr el servidor de node.js con el comando 

" npm start "  desde a terminal de visual. 

( Tener presente la ubicacion de la carpte PROYECTO_BACK_CELULARES en su pc )

Una vez el servidor de node este corriendo abrimos una nueva terminal , y escribimos lo siguiente : 

" cd auth-front " esto nos llevara a la carpeta descrita y desde alli corremos el otro servidor usando : 

" npm run dev" 

Cuando estos dos servidores esten corriendo se puede hacer uso de todas las funcionalidades de la pagina web.
