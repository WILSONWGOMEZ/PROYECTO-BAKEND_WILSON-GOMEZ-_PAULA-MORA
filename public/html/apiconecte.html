Introducción

Esta API permite gestionar una lista de productos (celulares) almacenados en una base de datos. Las operaciones disponibles incluyen obtener todos los productos, obtener un producto específico por su ID, crear un nuevo producto, actualizar un producto existente y eliminar un producto.

La estructura de cada producto es la siguiente:

{
    "_id": "66ee526b222c4bf59342a680",
    "id": 12,
    "brand": "Motorola",
    "price": 799.99,
    "storage": "256GB",
    "camera": "50MP",
    "connect": "5G",
    "ram": "12GB",
    "screen": "AMOLED",
    "warranty": "2years",
    "image_url": "https://i.blogs.es/602692/motorola-edge-30-pro-trasera/450_1000.jpg",
    "title": "Motorola Edge 30 Pro",
    "description": "El Motorola Edge 30 Pro ofrece un excelente rendimiento con su procesador Snapdragon 8 Gen 1 y una pantalla OLED de 6.7 pulgadas."
}

Endpoints de la API

GET /api/product

Descripción: Obtiene una lista de todos los productos disponibles en la base de datos.

Método HTTP: GET

URL: http://localhost:3000/api/product

Ejemplo de Solicitud

GET /api/product HTTP/1.1
Host: localhost:3000

Ejemplo de Respuesta Exitosa (200)

{
    "products": [
        {
            "_id": "66ee526b222c4bf59342a680",
            "id": 12,
            "brand": "Motorola",
            "price": 799.99,
            "storage": "256GB",
            "camera": "50MP",
            "connect": "5G",
            "ram": "12GB",
            "screen": "AMOLED",
            "warranty": "2years",
            "image_url": "https://i.blogs.es/602692/motorola-edge-30-pro-trasera/450_1000.jpg",
            "title": "Motorola Edge 30 Pro",
            "description": "El Motorola Edge 30 Pro ofrece un excelente rendimiento con su procesador Snapdragon 8 Gen 1 y una pantalla OLED de 6.7 pulgadas."
        },
        // ... otros productos
    ]
}

Respuesta si No Existen Productos (404)

{
    "message": "No existen productos."
}

GET /api/product/:productId

Descripción: Obtiene los detalles de un producto específico utilizando su ID único.

Método HTTP: GET

URL: http://localhost:3000/api/product/{productId}

Parámetros de Ruta

Host: localhost:3000

        "_id": "66ee526b222c4bf59342a680",
        "id": 12,
        "brand": "Motorola",
        "price": 799.99,
        "storage": "256GB",

        "connect": "5G",
        "ram": "12GB",
        "screen": "AMOLED",
        "warranty": "2years",
        "image_url": "https://i.blogs.es/602692/motorola-edge-30-pro-trasera/450_1000.jpg",

        "description": "El Motorola Edge 30 Pro ofrece un excelente rendimiento con su procesador Snapdragon 8 Gen 1 y una pantalla OLED de 6.7 pulgadas."

{
    "message": "El producto no existe."

Descripción: Crea un nuevo producto en la base de datos.

Método HTTP: POST

URL: http://localhost:3000/api/product

Cabeceras de la Solicitud

  • Content-Type: application/json

Cuerpo de la Solicitud (JSON)

{
    "id": 13,
    "brand": "Samsung",
    "price": 999.99,
    "storage": "512GB",
    "camera": "108MP",
    "connect": "5G",
    "ram": "16GB",
    "screen": "Dynamic AMOLED",
    "warranty": "2years",
    "image_url": "https://example.com/samsung-galaxy-s21.jpg",
    "title": "Samsung Galaxy S21",
    "description": "El Samsung Galaxy S21 ofrece un rendimiento excepcional con su procesador Exynos 2100 y una pantalla Dynamic AMOLED de 6.2 pulgadas."
}

PUT /api/product/:productId

Descripción: Actualiza los detalles de un producto existente en la base de datos.

Método HTTP: PUT

URL: http://localhost:3000/api/product/{productId}

Parámetros de Ruta

  • productId: ID único del producto a actualizar.

Cabeceras de la Solicitud

  • Content-Type: application/json

Cuerpo de la Solicitud (JSON)

Incluye solo los campos que deseas actualizar. Por ejemplo, para actualizar el precio y la garantía:

{
    "price": 899.99,
    "warranty": "3years"
}

Ejemplo de Solicitud

PUT /api/product/66ee526b222c4bf59342a680 HTTP/1.1
Host: localhost:3000
Content-Type: application/json

{
    "price": 899.99,
    "warranty": "3years"
}

Ejemplo de Respuesta Exitosa (200)

{
    "product": {
        "_id": "66ee526b222c4bf59342a680",
        "id": 12,
        "brand": "Motorola",
        "price": 899.99,
        "storage": "256GB",
        "camera": "50MP",
        "connect": "5G",
        "ram": "12GB",
        "screen": "AMOLED",
        "warranty": "3years",
        "image_url": "https://i.blogs.es/602692/motorola-edge-30-pro-trasera/450_1000.jpg",
        "title": "Motorola Edge 30 Pro",
        "description": "El Motorola Edge 30 Pro ofrece un excelente rendimiento con su procesador Snapdragon 8 Gen 1 y una pantalla OLED de 6.7 pulgadas.",
        "__v": 0
    }
}

Respuesta si el Producto No Existe (404)

{
    "message": "El producto no existe."
}

Respuesta si Hay un Error al Actualizar (500)

{
    "message": "Error al actualizar el producto: [detalle del error]"
}

DELETE /api/product/:productId

Descripción: Elimina un producto específico de la base de datos utilizando su ID único.

Método HTTP: DELETE

URL: http://localhost:3000/api/product/{productId}

Parámetros de Ruta

  • productId: ID único del producto a eliminar.

Ejemplo de Solicitud

DELETE /api/product/66ee526b222c4bf59342a680 HTTP/1.1
Host: localhost:3000

Ejemplo de Respuesta Exitosa (200)

{
    "message": "El producto ha sido eliminado"
}

Respuesta si el Producto No Existe (404)

{
    "message": "El producto no existe"
}

Respuesta si Hay un Error al Eliminar (500)

{
    "message": "Error al borrar el producto: [detalle del error]"
}
